# SDZ Modding — Home Page

Official public website for SDZ Modding.

## Routes

- `/` — landing page
- `/licensing` — SDZ licensing overview
- `/legal` — legal documentation
- `/info`, `/mods`, `/modhub` — reserved for future public pages

The prominent login action links to the existing SDZ ModHub account page.

## Development

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

Verify a production build with:

```bash
npm run build
```
